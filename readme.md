<p><span style="font-size:18px">Difficulty Lock is a minimalistic configurable mod which adds an easy way to configure the difficulty in all (new) worlds. Can be useful for modpacks, or just in your singleplayer experience if you have a default difficulty you always play with.&nbsp;</span><br><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/ricksouth/serilum-mc-mods/wiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong><span style="font-size:11px"><em>The priority shows how the mod handles when multiple gamemodes are set to true in the config. E.g. if all are enabled all new worlds will be peaceful.</em></span><strong><br>forcePeaceful</strong>&nbsp;(default = false): Priority 1: Sets the difficulty in any world to peaceful when enabled.<br><strong>forceEasy</strong>&nbsp;(default = false): Priority 2: Sets the difficulty in any world to easy when enabled.<br><strong>forceNormal</strong>&nbsp;(default = false): Priority 3: Sets the difficulty in any world to normal when enabled.<br><strong>forceHard</strong>&nbsp;(default = true): Priority 4: Sets the difficulty in any world to hard when enabled.<br><br><strong>shouldLockDifficulty</strong>&nbsp;(default = true): When enabled, locks the difficulty in any world so it cannot be changed.<br><strong>shouldChangeDifficultyWhenAlreadyLocked</strong>&nbsp;(default = false): When enabled, also sets the difficulty in worlds where it has already been locked.<br><br><br><span style="font-size:14px"><strong>With the default configuration, all worlds will be set to hard with the difficulty locked:</strong></span><br><picture><img src="https://github.com/ricksouth/serilum-mc-mods/raw/master/cdn/difficulty-lock/a.png" width="1142" height="597"></picture></p>
